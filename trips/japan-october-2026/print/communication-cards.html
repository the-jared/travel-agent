Communication Cards

Print on A4, cut along card borders. Cards fit in a wallet or passport holder.

2. Allergy / Dietary Restriction Card

> Customize this card by checking the relevant items. Show to restaurant staff before ordering.

```

┌─────────────────────────────────────────────────┐

│ │

│ 食物アレルギーがあります。 │

│ I have food allergies. │

│ │

│ 以下の食品が食べられません: │

│ I cannot eat the following: │

│ │

│ □ エビ・カニ (甲殻類) Shrimp / Crab │

│ □ 小麦 Wheat / Gluten │

│ □ 卵 Eggs │

│ □ 乳製品 Dairy / Milk │

│ □ そば Buckwheat / Soba │

│ □ 落花生・ナッツ Peanuts / Tree Nuts │

│ □ 魚 Fish │

│ □ 大豆 Soy │

│ □ 貝類 Shellfish │

│ □ 肉類 Meat (all) │

│ □ 豚肉 Pork │

│ □ 牛肉 Beef │

│ │

│ アレルギー反応は重篤です。 │

│ My allergy is severe. │

│ │

│ □ はい (Yes) □ いいえ (No, preference only) │

│ │

│ ご確認ありがとうございます。 │

│ Thank you for checking. │

│ │

└─────────────────────────────────────────────────┘

```

Vegetarian / Vegan Card

```

┌─────────────────────────────────────────────────┐

│ │

│ ベジタリアンです。 │

│ I am vegetarian. │

│ │

│ 肉と魚は食べられません。 │

│ I cannot eat meat or fish. │

│ │

│ 出汁（かつお出汁）も食べられません。 │

│ I also cannot eat dashi (bonito fish stock). │

│ │

│ 卵と乳製品は大丈夫です。 │

│ Eggs and dairy are OK. │

│ (Cross out if vegan) │

│ │

│ よろしくお願いします。 │

│ Thank you for your understanding. │

│ │

└─────────────────────────────────────────────────┘

```

Note: Dashi (fish stock) is in almost everything in Japanese cuisine, including many "vegetable" dishes. This card specifically flags it.

4. Top 20 Restaurant Phrases

| # | English | Japanese | Pronunciation |

|---|---------|----------|---------------|

| 1 | Table for 2, please | 二名です | Ni-mei desu |

| 2 | Do you have an English menu? | 英語のメニューはありますか？ | Eigo no menyuu wa arimasu ka? |

| 3 | What do you recommend? | おすすめは何ですか？ | Osusume wa nan desu ka? |

| 4 | I'll have this one (pointing) | これをお願いします | Kore wo onegaishimasu |

| 5 | One more of this, please | これをもう一つお願いします | Kore wo mou hitotsu onegaishimasu |

| 6 | Water, please | お水をお願いします | Omizu wo onegaishimasu |

| 7 | Beer, please | ビールをお願いします | Biiru wo onegaishimasu |

| 8 | Check, please | お会計をお願いします | Okaikei wo onegaishimasu |

| 9 | It was delicious! | おいしかったです！ | Oishikatta desu! |

| 10 | Thank you for the meal | ごちそうさまでした | Gochisousama deshita |

| 11 | I have a reservation | 予約があります | Yoyaku ga arimasu |

| 12 | Is this seat taken? | この席は空いていますか？ | Kono seki wa aite imasu ka? |

| 13 | No wasabi, please | わさび抜きでお願いします | Wasabi nuki de onegaishimasu |

| 14 | Is this spicy? | これは辛いですか？ | Kore wa karai desu ka? |

| 15 | Can I pay by card? | カードで払えますか？ | Kaado de haraemasu ka? |

| 16 | This is not what I ordered | 注文と違います | Chuumon to chigaimasu |

| 17 | To-go, please | テイクアウトでお願いします | Teiku auto de onegaishimasu |

| 18 | I don't eat meat | 肉は食べられません | Niku wa taberaremasen |

| 19 | Does this contain [X]? | これに[X]は入っていますか？ | Kore ni [X] wa haitte imasu ka? |

| 20 | Excuse me (to call staff) | すみません | Sumimasen |

Usage notes:

• Say "sumimasen" to get a server's attention (not rude, it's expected)

• "Gochisousama deshita" when leaving — say it to the chef/staff. Shows appreciation.

• Most restaurants have a call button on the table — press it instead of waving

Digital Alternatives

• Google Translate app: Download Japanese offline pack before the trip. Use camera mode to translate menus in real-time.

• Papago app: Often better than Google for Japanese. Download offline pack.

• Google Lens: Point at any sign or menu for instant translation.

Tip: Even with translation apps, having printed/saved cards is faster and more polite in a restaurant setting. Pulling out a card shows you prepared and respect the effort.
