# Visual & Image Research Guide

## Core Principle

A picture is worth a thousand words. Every recommended place should include
visual context so the traveler knows what to expect and can make informed choices.

---

## Image Sources

### For Every Recommended Location, Search:
1. **Google Images:** "[place name] [city]" — recent photos
2. **Google Maps Photos:** Click on the place, view user-submitted photos
3. **Instagram:** Location tag for the place — shows real visitor photos
4. **YouTube Thumbnails:** "[place name] tour/visit" — video walkthroughs
5. **Official Website:** Press/media section often has high-quality images

### What to Document Visually
For each recommendation, note:
- **What it actually looks like** (vs. marketing photos)
- **Crowd levels** (check recent Google Maps photos for real crowds)
- **Seasonal appearance** (cherry blossoms vs. bare trees)
- **Dress code visual** (what are people wearing in photos?)
- **Instagram vs. Reality** (flag if a place is heavily filtered/staged)

## Image Reference Format in Deliverables

Since the skill outputs markdown files, reference images as:

```
#### [Place Name]
**Visual preview:** Search "[place name] [city]" on Google Images
**Video walkthrough:** [YouTube link if found]
**Google Maps photos:** [Google Maps link — click "Photos" tab]
**Instagram location:** [Instagram location tag link]
**Street View:** [Google Street View link for approach/entrance]
```

### Google Street View Links
Generate Street View links for key locations:
`https://www.google.com/maps/@[lat],[lng],3a,75y,90t/data=!3m6!1e1!3m4!1s[panoID]!2e0!7i16384!8i8192`

Or simpler: include the Google Maps link and note "Click Street View for preview"

## Photo Opportunity Planning

### Sunrise / Sunset Spots
For each destination, research and include:
- [ ] Top 3 sunrise viewpoints (with exact time for trip dates)
- [ ] Top 3 sunset viewpoints (with exact time for trip dates)
- [ ] Best time to photograph major landmarks (light direction, crowd levels)
- [ ] Blue hour spots (30 min after sunset — city lights)

## Video Research Protocol

### YouTube Search Queries (For Experiential Context)
- "[city] walking tour [year]" — see what the streets actually feel like
- "[city] food tour" — see the food being prepared and served
- "[city] hidden gems" — visual discovery of off-radar spots
- "[city] hotel room tour [hotel name]" — see rooms before booking
- "[city] vlog [year]" — real traveler experience, not marketing
- "[city] worst mistakes" / "things I wish I knew" — avoid pitfalls

### What to Extract From Videos
- Real walking distances and conditions
- Actual crowd levels at attractions
- Real food portions and presentation
- Hotel room actual size and condition
- Neighborhood vibe (noisy? calm? sketchy?)
- Transport experience (is the metro confusing? is the bus crowded?)
