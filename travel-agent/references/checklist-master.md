# Master Checklist Collection

## Instructions

Generate ALL applicable checklists from this master list, customized to the
specific trip. Remove items that don't apply. Add destination-specific items.

---

## 1. Pre-Trip Planning Checklist (4-8 Weeks Before)

- [ ] Passport valid for 6+ months beyond return date
- [ ] Visa application submitted (if required)
- [ ] Flights booked
- [ ] Accommodation booked
- [ ] Travel insurance purchased
- [ ] Vaccination appointments scheduled (if needed)
- [ ] International driving permit obtained (if driving)
- [ ] Pet care arranged (if applicable)
- [ ] Plant watering arranged
- [ ] Mail hold / forwarding set up
- [ ] Credit card company notified of travel dates
- [ ] Bank notified for international ATM use
- [ ] Check credit card foreign transaction fees
- [ ] Download offline maps (Google Maps / Maps.me)
- [ ] Download translation app with offline language pack
- [ ] Research and bookmark key restaurant reservations
- [ ] Book any advance-reservation activities (Michelin restaurants, popular tours)
- [ ] Register with embassy (US: STEP program, UK: equivalent)
- [ ] Photocopy passport, visas, insurance — store digitally and in luggage
- [ ] Check phone plan for international roaming / purchase eSIM
- [ ] Research airport lounge access (credit card perks?)
- [ ] Check luggage weight limits for all flights

## 2. One Week Before Checklist

- [ ] Check-in for flight online (24h before)
- [ ] Confirm all reservations (hotel, tours, restaurants)
- [ ] Check weather forecast and adjust packing
- [ ] Charge all electronics
- [ ] Download entertainment for flight (movies, podcasts, books)
- [ ] Print or download boarding passes
- [ ] Print or save hotel confirmation
- [ ] Exchange small amount of local currency for arrival
- [ ] Set up currency converter app
- [ ] Brief someone at home on your itinerary
- [ ] Set email out-of-office
- [ ] Pay upcoming bills that will be due while traveling
- [ ] Clean out fridge of perishables
- [ ] Take out trash
- [ ] Arrange airport transport

## 3. Packing — Universal Base

### Carry-On (Plane Essentials)
- [ ] Passport + visa documents
- [ ] Boarding pass
- [ ] Travel insurance card / policy number
- [ ] Wallet (local currency + cards)
- [ ] Phone + charger + portable battery
- [ ] Headphones / earbuds
- [ ] Neck pillow (if long flight)
- [ ] Eye mask + ear plugs
- [ ] Change of clothes (in case luggage delayed)
- [ ] Medications (prescription + basic first aid)
- [ ] Snacks
- [ ] Entertainment (book, tablet, downloaded shows)
- [ ] Pen (for immigration forms)
- [ ] Reusable water bottle (empty through security)
- [ ] Laptop / tablet + charger (if needed)
- [ ] Camera + memory card + charger (if separate from phone)
- [ ] Ziplock bags for liquids (3-1-1 rule)

### Checked Luggage — Clothing
Customize based on climate and activities:

#### Warm Climate (Beach / Tropical)
- [ ] Lightweight shirts/tops (×N days, plan to rewear)
- [ ] Shorts (×3-4)
- [ ] Lightweight pants/trousers (×1-2)
- [ ] Swimsuit(s) (×2)
- [ ] Sarong / cover-up
- [ ] Light jacket / cardigan (AC in restaurants)
- [ ] Sandals / flip-flops
- [ ] Walking shoes (comfortable, broken-in)
- [ ] Water shoes (rocky beaches)
- [ ] Sun hat
- [ ] Sunglasses
- [ ] Rain jacket (packable)

#### Cold Climate (Winter / Mountain)
- [ ] Base layers (thermal top + bottom)
- [ ] Mid layers (fleece / down jacket)
- [ ] Outer shell (waterproof jacket)
- [ ] Warm pants (×2-3)
- [ ] Long-sleeve shirts (×N)
- [ ] Sweaters (×2)
- [ ] Warm hat / beanie
- [ ] Gloves (waterproof + liner)
- [ ] Scarf / neck gaiter
- [ ] Warm socks (wool, ×N)
- [ ] Waterproof boots
- [ ] Indoor shoes (for heated spaces)

#### Temperate / Shoulder Season
- [ ] Mix of layers
- [ ] Rain jacket essential
- [ ] Umbrella (compact)
- [ ] Mix of shorts and pants
- [ ] Light sweater / hoodie
- [ ] Comfortable walking shoes
- [ ] One dressy outfit (for nice dinners)

## 4. Activity-Specific Checklists

### Hiking / Trekking
- [ ] Hiking boots (broken in!)
- [ ] Hiking socks (wool, moisture-wicking)
- [ ] Daypack
- [ ] Water purification (tablets or filter)
- [ ] Trail snacks / energy bars
- [ ] Headlamp
- [ ] Trekking poles (if needed)
- [ ] Blister kit (moleskin)
- [ ] Rain cover for pack

### Beach / Snorkeling / Diving
- [ ] Reef-safe sunscreen
- [ ] Rash guard
- [ ] Snorkel mask (own for hygiene — or rent)
- [ ] Dive certification card (PADI/SSI)
- [ ] Dive log book
- [ ] Underwater camera / housing
- [ ] Waterproof bag for electronics

### Cold Weather / Skiing
- [ ] Ski pass
- [ ] Ski gear (rent or bring)
- [ ] Goggles
- [ ] Hand/toe warmers
- [ ] Sunscreen (reflection off snow)
- [ ] Lip balm (critical)

### Business / Conference
- [ ] Business cards
- [ ] Laptop + charger
- [ ] Presentation backup on USB
- [ ] Business attire
- [ ] Portfolio / padfolio

### Photography-Focused
- [ ] Camera body + lenses
- [ ] Extra batteries
- [ ] Extra memory cards
- [ ] Lens cleaning kit
- [ ] Tripod
- [ ] Drone (check local laws!)
- [ ] Laptop for editing (if trip is long)
- [ ] Hard drive for backup

## 5. Day-of-Departure Checklist

- [ ] Water plants
- [ ] Empty trash
- [ ] Unplug non-essential electronics
- [ ] Set home security system
- [ ] Lock all windows and doors
- [ ] Leave key with trusted person
- [ ] Thermostat adjusted
- [ ] Check passport and wallet one final time
- [ ] Check flight status (delays?)
- [ ] Arrive at airport: domestic 2h, international 3h before

## 6. Arrival Day Checklist

- [ ] Get through immigration (have docs ready)
- [ ] Collect luggage
- [ ] Purchase SIM card / activate eSIM
- [ ] Get local currency from ATM (avoid airport exchange)
- [ ] Find transport to hotel (per transport plan)
- [ ] Check into accommodation
- [ ] Test WiFi
- [ ] Identify nearest: pharmacy, convenience store, ATM, restaurant
- [ ] Walk the immediate neighborhood
- [ ] Adjust to time zone (get sunlight, stay up until local bedtime)

## 7. Return Home Checklist

- [ ] Check out of accommodation (nothing left behind — check safe, drawers, bathroom)
- [ ] VAT/tax refund at airport (if eligible)
- [ ] Use remaining local currency (or save for future trip)
- [ ] Duty-free shopping (if desired)
- [ ] Delete sensitive data from devices (hotel WiFi passwords, etc.)
- [ ] Back up photos
- [ ] Leave review for accommodation
- [ ] Cancel any local SIM/data plans
- [ ] Reclaim any deposits
