# Real-Time Monitoring & Trip Adjustment Guide

## Pre-Trip Monitoring (Set Up Before Departure)

### Flight Monitoring
- [ ] Google Flights — Track booked flights for schedule changes
- [ ] Flighty app — Real-time flight tracking with gate changes, delays
- [ ] Airline app — Push notifications for flight status
- [ ] FlightAware — Detailed flight tracking and delay predictions

### Weather Monitoring
- [ ] Weather.com — 10-day forecast for destination
- [ ] Windy.com — Advanced weather visualization
- [ ] AccuWeather — Hour-by-hour forecast
- [ ] Rain radar apps — Real-time precipitation tracking

### Safety Monitoring
- [ ] STEP enrollment (US citizens) — Embassy alerts
- [ ] Google Alerts — "[destination] travel advisory"
- [ ] Twitter/X lists — Follow @USEmbassy[Country], local news
- [ ] Smartraveller (Australia) / Gov.uk (UK) — Advisory updates

### Price Monitoring
- [ ] Google Flights — Price drop alerts on booked routes
- [ ] Pruvo — Automatic hotel price drop monitoring
- [ ] Hopper — Price prediction updates
- [ ] CamelCamelCamel — For gear/equipment purchases

## During-Trip Adjustments

### Daily Morning Check (2 Minutes)
1. Check weather for today and tomorrow
2. Check any reservation confirmations
3. Verify transit running normally (Google Maps real-time)
4. Glance at news for destination (any events/disruptions?)

### Common Adjustments & Solutions

| Situation | Immediate Action | Tool |
|---|---|---|
| Rain forecast | Swap outdoor activity for indoor | Plan B file |
| Restaurant closed | Check backup in alternatives | Plan B file |
| Transit disruption | Route via alternative transit | Google Maps real-time |
| Flight delayed | Check rebooking options in airline app | Airline app + insurance |
| Attraction closed | Visit nearby alternative | Activities file |
| Feeling unwell | Rest day, locate pharmacy | Safety file |
| Budget overspend | Switch to budget tier for remaining days | Budget file |
| Found amazing spot | Extend stay, cut a planned activity | Itinerary file |

### Emergency Protocol

```
IF medical emergency:
  1. Call [local emergency number]
  2. Call hotel front desk [number]
  3. Go to nearest hospital [address from safety file]
  4. Call insurance hotline [number]
  5. Call embassy [number]
  6. Contact emergency contact at home [number]

IF theft/robbery:
  1. Get to safe location
  2. Call tourist police [number]
  3. File police report (needed for insurance)
  4. Cancel stolen credit cards [numbers in emergency card]
  5. Contact embassy if passport stolen [number]
  6. Call insurance hotline [number]

IF natural disaster/political unrest:
  1. Follow local authority instructions
  2. Contact embassy [number]
  3. Check STEP enrollment alerts
  4. Stay in safe location with supplies
  5. Contact emergency contact at home
  6. Monitor local news (in English: [news source])
```
