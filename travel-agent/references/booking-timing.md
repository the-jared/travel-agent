# Booking Timing Intelligence

## Core Principle

Timing is money. The difference between booking at the right time vs. the
wrong time can be 30-60% on flights and 20-40% on hotels. This guide ensures
every trip plan includes precise booking window recommendations.

---

## Flight Booking Windows

### Domestic Flights
- **Sweet spot:** 1-3 months before departure
- **Too early:** 6+ months (prices haven't settled)
- **Too late:** Under 2 weeks (prices spike)
- **Best day to book:** Tuesday-Wednesday (marginal, but real)
- **Best day to fly:** Tuesday, Wednesday, Saturday (cheapest)

### International Flights (Short-Haul, Under 6h)
- **Sweet spot:** 2-4 months before departure
- **Budget airlines:** Book as early as possible (prices only go up)
- **Legacy carriers:** Wait for sales, set price alerts

### International Flights (Long-Haul, 6h+)
- **Sweet spot:** 3-6 months before departure
- **Peak season (summer/holidays):** Book 5-8 months ahead
- **Premium economy/business:** Watch for sales 4-6 months out
- **Last-minute deals:** Rare for long-haul, don't count on them

### Peak vs. Off-Peak Pricing

| Season | Flight Premium | Hotel Premium | When |
|---|---|---|---|
| Peak | +40-80% | +50-100% | Jun-Aug, Dec holidays, local holidays |
| Shoulder | +10-20% | +10-30% | Apr-May, Sep-Oct |
| Off-peak | Baseline | Baseline | Nov, Jan-Mar (non-holiday) |

### Price Monitoring Tools
- **Google Flights:** Price tracking with email alerts
- **Hopper:** Predicts "buy now" vs "wait" with confidence %
- **Skyscanner:** Price alerts + "cheapest month" view
- **Kayak:** Price forecast (green/red confidence)
- **Secret Flying / The Points Guy:** Error fares and sales

## Hotel Booking Windows

### Standard Hotels
- **Sweet spot:** 2-4 weeks before (best selection + good prices)
- **Strategy:** Book refundable NOW, keep monitoring, rebook if cheaper
- **Free cancellation:** Always choose this, rebook if price drops
- **Non-refundable:** Only for confirmed dates, 15-25% cheaper

### Peak Season / Events
- **Book 3-6 months ahead** if traveling during:
  - Cherry blossom season (Japan: late March-April)
  - Oktoberfest (Munich: Sep-Oct)
  - New Year's Eve (any major city)
  - Carnival (Rio, Venice: Feb-Mar)
  - Major conferences/events

### Airbnb
- **Sweet spot:** 2-6 weeks ahead for most cities
- **Superhost premium properties:** Book 1-2 months ahead
- **Long stays (28+ nights):** Significant monthly discounts
- **New listings:** Often discounted 20% to get initial reviews

### Boutique / Luxury Hotels
- **Book early:** Popular properties sell out months ahead
- **Direct booking perks:** Free upgrade, late checkout, welcome drink
- **Loyalty programs:** Sign up before booking for instant benefits

## Activity Booking Windows

| Activity Type | When to Book | Why |
|---|---|---|
| Michelin restaurants | 2-3 months ahead | Releases by month, sells out instantly |
| Popular museum timed entry | 2-4 weeks ahead | Slots fill (Uffizi, Anne Frank, Alhambra) |
| Cooking classes | 1-2 weeks ahead | Small groups fill fast |
| Day tours | 3-7 days ahead | Usually available, but popular ones fill |
| Adventure activities | 1-2 weeks ahead | Weather-dependent, need confirmation |
| Theater/shows | 1-3 months ahead | Popular shows (West End, Broadway) sell out |
| Festival tickets | As soon as announced | Often sell out in hours/days |
| Spa/wellness | 1 week ahead | Weekends fill faster |
| Walking tours (free) | Day-of | Just show up |
| Skip-the-line passes | 1-2 weeks ahead | Save 1-2 hours in queue |

## Booking Timeline Template

For each trip, generate a countdown:

```
## Booking Timeline — [Trip Name]

### 6 Months Before ([Date])
- [ ] Book flights (if peak season)
- [ ] Book must-stay hotel (if peak/event)
- [ ] Book Michelin restaurants

### 3 Months Before ([Date])
- [ ] Book flights (if not done)
- [ ] Book primary accommodation
- [ ] Book popular tours/activities
- [ ] Book theater/show tickets
- [ ] Apply for visa (if required)

### 1 Month Before ([Date])
- [ ] Confirm all reservations
- [ ] Book remaining restaurants
- [ ] Book cooking classes / experiences
- [ ] Purchase transit passes online (if available)
- [ ] Set up price alerts for any unbought flights

### 2 Weeks Before ([Date])
- [ ] Book airport transfers
- [ ] Book any remaining activities
- [ ] Check-in for flight (when window opens)
- [ ] Download offline maps
- [ ] Purchase eSIM / activate roaming

### 1 Week Before ([Date])
- [ ] Confirm all reservations (email hotels)
- [ ] Check weather forecast, adjust packing
- [ ] Print/save all confirmations
- [ ] Exchange small amount of currency
- [ ] Share itinerary with emergency contact

### Day Before ([Date])
- [ ] Online check-in for flight
- [ ] Charge all devices
- [ ] Final packing check against list
- [ ] Set alarm for departure time
- [ ] Check flight status
```

## Price Drop Strategy

If a booked refundable item drops in price:
1. Book the cheaper rate
2. Cancel the original booking
3. Save the confirmation number

Automate this with:
- **Pruvo:** Monitors hotel bookings for price drops and rebooks
- **Google Flights track:** Alerts you to price drops on booked routes
- **Credit card price protection:** Some cards refund the difference
