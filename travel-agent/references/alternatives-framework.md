# Alternatives & Plan B Framework

## Core Principle

Every plan needs a backup. For every recommendation, provide at least one
alternative. For every day in the itinerary, provide a rainy-day and a
low-energy alternative. The traveler should never be stuck without options.

---

## Alternative Layers

### 1. Accommodation Alternatives
For every recommended property, include:
- **Similar but different neighborhood:** [Name] — $X/night — [why this works]
- **Different tier:** [Name] — $X/night — [budget/luxury swap]
- **Last-minute fallback:** [Chain hotel/hostel] — [always available, no reservation needed]
- **Emergency:** [What to do if everything is fully booked]

### 2. Restaurant Alternatives
For every recommended restaurant:
- **Same cuisine, different spot:** [Name] — [walk-in friendly, no reservation needed]
- **Dietary alternative:** [Name] — [better for allergies/restrictions]
- **Budget swap:** [Name] — [similar vibe, half the price]
- **Closed/full fallback:** [Name] — [backup within 5-minute walk]

### 3. Activity Alternatives
For every planned activity:
- **If closed/sold out:** [Alternative activity in same area]
- **If raining:** [Indoor alternative nearby]
- **If too tired:** [Low-energy alternative]
- **If too crowded:** [Same activity, different time/location]
- **If running early:** [Quick add-on nearby]
- **If running late:** [What to cut, what to keep]

### 4. Transport Alternatives
For every transport leg:
- **If flight cancelled:** [Next flight, alternative airline, train option]
- **If train delayed:** [Bus alternative, taxi cost for the route]
- **If Uber/Grab unavailable:** [Taxi rank location, bus option, walking time]
- **If transit strike:** [Common in Europe — what to do]

### 5. Day-Level Alternatives
For EACH day in the itinerary, include a complete Plan B:

```
### Day [N] — Plan B (Rainy Day / Low Energy)

**If weather is bad:**
- Morning: [Indoor activity] — [location, cost]
- Lunch: [Cozy restaurant] — [covered seating, comfort food]
- Afternoon: [Museum / spa / shopping mall] — [location]
- Evening: [Original dinner plan still works / alternative]

**If energy is low:**
- Sleep in, skip morning activity
- Brunch at [Name] — [leisurely, close to hotel]
- Afternoon: [Gentle activity — park, cafe, easy walk]
- Combine tomorrow's must-do with today's leftover

**If plans change completely:**
- Unplanned day option A: [Day trip to X]
- Unplanned day option B: [Neighborhood deep-dive — Y area]
- Unplanned day option C: [Wellness — spa/onsen/hammam]
```

### 6. Trip-Level Contingencies
- **Missed connection:** [Airline rebooking rights, airport hotel options]
- **Lost luggage:** [Nearest clothing stores, what insurance covers]
- **Lost passport:** [Nearest embassy, emergency travel document process]
- **Medical emergency:** [Hospital, insurance hotline, pharmacy locations]
- **Natural disaster/political event:** [Embassy contact, evacuation routes]
- **Credit card blocked:** [Backup card, emergency cash, Western Union]
- **Phone lost/broken:** [Where to buy replacement, access accounts from computer]

## Output: Alternatives Summary

Every deliverable file should end with an "Alternatives" section:

```
## Alternatives & Backups

| Original Plan | If Unavailable | If Weather Bad | Budget Swap |
|---|---|---|---|
| [Restaurant A] | [Restaurant B] | [Restaurant C — covered] | [Street food D] |
| [Temple visit] | [Different temple] | [Museum nearby] | [Free park] |
| [Beach day] | [Different beach] | [Indoor market] | [City walk] |
```
