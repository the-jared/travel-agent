# Transport Analysis Methodology

## Research Scope

Cover EVERY leg of transport: origin → airport → destination airport → city →
between cities → between attractions → return.

---

## 1. Flights

### Search Protocol
- **Google Flights** — Primary comparison. Check flexible dates (+/- 3 days).
- **Skyscanner** — "Everywhere" search for open-ended destinations.
- **Kayak** — Price alerts and trend predictor ("buy now" vs "wait").
- **Direct airline sites** — Often cheaper, better seats, easier changes.
- **Budget airlines** — Check Ryanair, EasyJet, AirAsia, Peach, etc. separately (not always on aggregators).
- **Points/miles** — Note if booking with points is available.

### Present Per Route

```
### [Origin] → [Destination]
**Best value:** [Airline] — $X (economy) / $X (premium) / $X (business)
  - Duration: Xh Xm | Stops: [nonstop / 1 stop via City]
  - Departure: [time] → Arrival: [time local]
  - Baggage: [included? carry-on only? cost for checked?]
  - Seat pitch: [X inches]

**Best schedule:** [Different airline if applicable]

**Best comfort:** [Different airline if applicable]

**Booking window:** [Price likely to go up/down? Book now or wait?]
**Flexibility tip:** [Moving dates by 1-2 days saves $X]
```

## 2. Airport Transfers

For EVERY airport arrival, document:
- [ ] Taxi: Cost, duration, reliability, scam risks
- [ ] Private transfer: Cost, pre-booking options
- [ ] Ride-hailing: Available? App name? Approximate cost?
- [ ] Airport express train/bus: Cost, duration, frequency, last service
- [ ] Public transit: Route, cost, duration, luggage friendliness
- [ ] Hotel shuttle: Available from chosen accommodations?
- [ ] Rental car: Pick-up location, cost, drive to city duration

## 3. Intercity Transport

### Search Each Option
- **Train:** National rail site, Rome2Rio, Seat61.com
  - Class options, reserved vs unreserved, scenic routes
  - Rail passes (JR Pass, Eurail, BritRail) — cost vs. point-to-point
- **Bus:** FlixBus, national carriers, Busbud
  - Overnight buses as accommodation saver?
- **Domestic flights:** Compare against train for time/cost
- **Rental car:** When driving is the best option (countryside, flexibility)
  - International license needed?
  - Road conditions and driving culture
  - Parking costs at destinations
  - Tolls
- **Ferry:** For island destinations or scenic routes
- **Private driver:** Cost for day trips, multi-day tours

## 4. Local Transit Within Cities

- [ ] Metro/subway system: Map, fare, day pass, tourist card
- [ ] Bus system: Useful routes, how to pay
- [ ] Tram/light rail: If available
- [ ] Ride-hailing: Which apps work? Average costs?
- [ ] Taxis: Metered? Negotiate? Starting fare?
- [ ] Bicycle rental / e-scooter: Available? Safe?
- [ ] Walking: Walkable city? Sidewalk quality? Safety?
- [ ] Water taxi / boat: If applicable (Venice, Bangkok, Istanbul)

## 5. Transit Passes & Cards

Research and recommend transit passes:
- Tourist-specific passes (Tokyo Metro pass, London Oyster, Paris Navigo)
- All-in-one cards (attraction + transit bundles)
- Cost comparison: Pass vs. individual tickets for planned itinerary
- Where to buy (airport? online? convenience store?)
