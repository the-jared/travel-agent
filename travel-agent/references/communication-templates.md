# Communication Templates — Local Language

## Core Principle

Language barriers cause the most friction in travel. Pre-made messages and
printable cards in the local language eliminate that friction entirely.
The traveler should never struggle to communicate essential needs.

---

## Printable Cards (Generate for Every Trip)

### 1. Taxi Destination Card
A card the traveler shows to a taxi driver. Generate one per accommodation
and per frequently-visited destination.

```
┌─────────────────────────────────────────┐
│  DESTINATION / 目的地                     │
│                                         │
│  [Hotel Name in LOCAL SCRIPT]           │
│  [Full Address in LOCAL SCRIPT]         │
│                                         │
│  [Phone number of hotel]                │
│                                         │
│  📍 Google Maps: [short link]           │
└─────────────────────────────────────────┘
```

## Email/Message Templates

### Restaurant Reservation Request

```
Subject: Reservation / [LOCAL WORD for reservation]

[Greeting in local language],

I would like to make a reservation:
- Date: [Date]
- Time: [Time]
- Number of guests: [N]
- Any allergies: [List]
- Special requests: [Window seat, birthday, etc.]

Name: [Full Name]
Phone: [Number]
Email: [Email]

[Thank you in local language],
[Name]
```

### Hotel Special Request Email

```
Subject: Guest Request — Confirmation #[Number]

Dear [Hotel Name],

I have a reservation under [Name], confirmation #[Number],
checking in [Date].

I would like to request:
- [Early check-in / Late checkout / Airport transfer / etc.]
- [Room preference — high floor, quiet, view, etc.]
- [Special occasion — anniversary, birthday, honeymoon]

Thank you for your help.

Best regards,
[Name]
```

### Cancellation/Change Request

```
Subject: Change Request — Booking #[Number]

Dear [Provider],

I need to [change/cancel] my booking:
- Booking reference: [Number]
- Original date: [Date]
- New date: [Date] (if changing)
- Reason: [Brief reason]

Please confirm the [change/cancellation] and any applicable fees.

Thank you,
[Name]
```
