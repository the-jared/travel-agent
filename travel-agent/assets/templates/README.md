# Output Templates Index

Templates used by the synthesizer to generate trip deliverable files.
Each template defines the exact structure, sections, and formatting
for its corresponding output file.

| Template | Output File | Purpose |
|---|---|---|
| `itinerary-template.md` | `01-itinerary.md` | Day-by-day schedule with time blocks |
| `comparison-table.md` | Used in `03`, `04`, `10` | Side-by-side option comparisons |
| `country-guide-template.md` | `02-country-guide.md` | Full country knowledge guide |
| `packing-list-base.md` | `09-checklists.md` (packing section) | Climate-adaptive packing |
| `step-by-step-guide.md` | `15-step-by-step-guides.md` | Transit directions format |
| `daily-pocket-card.md` | `12-pocket-guide.md` | Phone-friendly daily card |
| `trip-summary-card.md` | `00-overview.md` | One-page trip overview |
